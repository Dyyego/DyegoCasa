Lenovo IdeaPad Gaming 3i

Conheça o poderoso notebook para jogos da Lenovo

Visão Geral

O Lenovo IdeaPad Gaming 3i é uma excelente opção de laptop para entusiastas de jogos que buscam desempenho e qualidade.

Variantes

  • IdeaPad Gaming 3i Base: Processador Intel Core i5, GPU NVIDIA GTX 1650, 8GB de RAM, 512GB SSD.
  • IdeaPad Gaming 3i Pro: Processador Intel Core i7, GPU NVIDIA RTX 3050, 16GB de RAM, 1TB SSD.

Especificações Gerais

  • Processador: Intel Core i5 / i7
  • Placa de Vídeo: NVIDIA GTX 1650 / RTX 3050
  • Memória RAM: 8GB / 16GB
  • Armazenamento: 512GB SSD / 1TB SSD
  • Tela: 15.6 polegadas Full HD
  • Sistema Operacional: Windows 10

Recursos Adicionais

  • Teclado retroiluminado
  • Áudio Dolby Atmos
  • Conexões: USB-C, USB-A, HDMI, Ethernet

Preços

Os preços variam de acordo com a configuração escolhida, com preços iniciando em R$ 4.000,00 para o modelo base.
